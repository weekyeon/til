# VSCode Shortcut

* [Visual Studio Code tutorial, 단축키](https://demun.github.io/vscode-tutorial/shortcuts/)



### 내가 자주 사용하는 단축키

| 명령                       | Key                  |
| -------------------------- | -------------------- |
| 한 줄 삭제                 | Shift + Ctrl + K     |
| 한 줄 복사                 | Shift + Alt + ↓, ↑   |
| 한 줄 이동                 | Alt + ↓, ↑           |
| 위에 행 삽입               | Ctrl + Shift + Enter |
| 반복되는 코드 한 번에 수정 | Ctrl + D             |
| 변수명/함수명 한 번에 수정 | F2                   |
| 코드 일괄 주석 처리        | Ctrl + /             |
| 다중 커서 선택하여 수정    | Ctrl + Alt + ↓, ↑    |
| 코드 마지막에 커서 두기    | Shift + Alt + I      |
